# NAS 云同步配置指南

## 概述

NAS 同步功能允许您将游戏数据备份到自己的 NAS 设备或任何支持 WebDAV 的服务器上，无需依赖第三方云服务。

## 支持的 NAS 类型

- Synology NAS
- QNAP NAS
- TrueNAS
- 其他支持 WebDAV 的文件服务器

## 配置步骤

### 1. 启用 WebDAV 服务

#### Synology NAS
1. 打开 "控制面板"
2. 选择 "应用程序终端"
3. 勾选 "启用 WebDAV" 和 "HTTPS"（推荐）
4. 设置端口号（默认 5006 用于 HTTPS）
5. 选择用户权限

#### QNAP NAS
1. 打开 "控制面板"
2. 选择 "应用程序" → "Web 服务器"
3. 勾选 "启用 WebDAV"
4. 设置端口号

### 2. 创建同步文件夹

1. 在 File Station 中创建新文件夹，例如 `Sudoku`
2. 设置适当的访问权限
3. 记住文件夹路径（需要填写在应用配置中）

### 3. 配置应用

1. 打开数独应用
2. 进入 "设置"
3. 开启 "云同步"
4. 选择 "NAS / 自建服务器"
5. 填写配置信息：
   - **服务器地址**: `https://your-nas-ip:5006` 或 `http://your-nas-ip:5005`
   - **用户名**: NAS 登录用户名
   - **密码**: NAS 登录密码
   - **同步路径**: `/Sudoku` 或你创建的文件夹路径

6. 点击 "测试连接" 验证配置
7. 配置成功后，输入昵称和 PIN 码即可开始同步

## 数据存储

数据将以 JSON 格式保存到指定的 NAS 路径，文件名格式为：
```
sudoku_{用户ID}.json
```

其中用户ID由昵称和 PIN 码生成，确保不同玩家的数据不会冲突。

## 常见问题

### 连接失败
- 检查服务器地址是否正确
- 确认 WebDAV 服务已启用
- 验证用户名和密码
- 检查防火墙设置
- 确保使用 HTTPS（推荐）

### 权限问题
- 确认 NAS 用户对同步文件夹有读写权限
- 检查文件夹是否被其他应用程序锁定

### 跨设备同步
- 确保所有设备配置相同的服务器地址和路径
- 使用相同的昵称和 PIN 码登录

## 安全建议

1. **使用 HTTPS**: 建议启用 HTTPS 以加密数据传输
2. **强密码**: 为 NAS 用户设置强密码
3. **定期备份**: 定期备份 NAS 上的数据
4. **访问控制**: 限制 WebDAV 服务的访问 IP 范围

## 技术说明

- 协议: WebDAV (HTTP/HTTPS)
- 数据格式: JSON
- 认证: HTTP Basic Auth
- 无需额外软件依赖
